Getting Started

Prerequisites

DependencyVersionInstall
macOS14.0+--
Xcode16.0+developer.apple.com
zig0.15.2brew install zig
XcodeGenlatestbrew install xcodegen

Clone and Setup

# Clone with submodules (includes Ghostty fork)
git clone --recursive https://github.com/manaflow-ai/mosaic.git
cd mosaic

# Run setup script -- builds GhosttyKit xcframework
./Scripts/setup.sh

The setup script will:

  1. Initialize and update the Ghostty git submodule
  2. Verify that zig is installed
  3. Build GhosttyKit.xcframework with ReleaseFast optimization (cached by commit SHA)
  4. Symlink the framework into ghostty/zig-out/

The GhosttyKit build is cached at ~/.cache/mosaic/ghosttykit/ keyed by the Ghostty submodule commit SHA. Subsequent runs reuse the cache.

Building with Ghostty (Full Build)

# Generate the Xcode project from project.yml
xcodegen generate

# Build from command line
xcodebuild -scheme Mosaic -configuration Debug build

# Or open in Xcode and build with Cmd+R
open Mosaic.xcodeproj

Building without Ghostty (Stub Build)

For CI or development without the full Ghostty xcframework, the project includes stub headers in ghostty-stubs/ that provide the C API type declarations. This allows compilation and testing of non-FFI code. 

xcodegen generate
xcodebuild -scheme Mosaic -configuration Debug build

Running the App

After building, run the Mosaic scheme in Xcode or launch from the command line:

# Run from build products
open build/Debug/Mosaic.app

# Or use xcodebuild
xcodebuild -scheme Mosaic -configuration Debug build
open "$(xcodebuild -scheme Mosaic -configuration Debug -showBuildSettings \
    | grep ' BUILT_PRODUCTS_DIR' | awk '{print $3}')/Mosaic.app"

Running Tests

# Run all tests
xcodebuild -scheme Mosaic -configuration Debug test

# Run a specific test class
xcodebuild -scheme Mosaic -configuration Debug \
    -only-testing:MosaicTests/WorkspaceTests test

Socket integration tests in Tests/SocketTests/ are designed for CI only and should not be run locally.

Project Structure

mosaic/
  MosaicKit/         Core logic (no UI imports)
  MosaicUI/          SwiftUI + AppKit views
  MosaicGateway/     Telegram gateway server
  CLI/               mosaic command-line tool
  ghostty/           Ghostty submodule
  ghostty-stubs/     Stub C headers for builds without Ghostty
  Resources/         Info.plist, shell integration scripts
  Scripts/           setup.sh
  Tests/             Unit and integration tests
  project.yml        XcodeGen project definition

Verifying the Setup

After building and running, verify the following:

  1. A Mosaic window appears with a working terminal
  2. Typing produces characters with no visible lag
  3. The sidebar shows at least one workspace
  4. The socket is created at /tmp/mosaic.sock
  5. Run mosaic system ping from another terminal -- it should return pong: true
