CLI Reference

The mosaic CLI communicates with the running Mosaic app over a Unix domain socket using JSON-RPC 2.0. The default socket path is /tmp/mosaic.sock.

Usage

mosaic <namespace> <command> [--param value ...]

Global Flags

FlagDescriptionDefault
--jsonOutput raw JSON-RPC responseoff (pretty-printed)
--socket <path>Custom socket path/tmp/mosaic.sock
--timeout <secs>Request timeout in seconds5 (30 for long-running commands)

workspace.*

Manage workspaces (tabbed sessions). Only workspace select steals focus.

workspace list

List all workspaces with their IDs, titles, order, and selection state.

mosaic workspace list
mosaic workspace list --json

workspace create

Create a new workspace. Returns the new workspace ID.

mosaic workspace create --title "Backend"
mosaic workspace create  # uses default title "New Workspace"
ParamTypeRequiredDescription
--titlestringnoWorkspace name

workspace select

Switch to a workspace by ID. This command steals focus.

mosaic workspace select --id "A1B2C3D4-..."
ParamTypeRequiredDescription
--idUUIDyesWorkspace ID

workspace rename

Rename an existing workspace.

mosaic workspace rename --id "A1B2C3D4-..." --title "Frontend"
ParamTypeRequiredDescription
--idUUIDyesWorkspace ID
--titlestringyesNew title

workspace delete

Delete a workspace. Cannot delete the last remaining workspace.

mosaic workspace delete --id "A1B2C3D4-..."
ParamTypeRequiredDescription
--idUUIDyesWorkspace ID

pane.*

Manage panes within the active workspace. Only pane focus steals focus.

pane split

Split a pane horizontally or vertically. Creates a new terminal in the new pane.

mosaic pane split --direction horizontal
mosaic pane split --direction vertical --pane_id "A1B2C3D4-..."
ParamTypeRequiredDescription
--directionstringyeshorizontal or vertical
--pane_idUUIDnoPane to split (defaults to focused pane)

pane close

Close a pane.

mosaic pane close
mosaic pane close --pane_id "A1B2C3D4-..."
ParamTypeRequiredDescription
--pane_idUUIDnoPane to close (defaults to focused pane)

pane focus

Focus a specific pane. This command steals focus.

mosaic pane focus --pane_id "A1B2C3D4-..."
ParamTypeRequiredDescription
--pane_idUUIDyesPane to focus

pane resize

Resize a split by adjusting its ratio.

mosaic pane resize --split_id "A1B2C3D4-..." --ratio 0.6
ParamTypeRequiredDescription
--split_idUUIDyesSplit node ID
--ratiofloatyesSplit ratio (0.05 to 0.95)

pane send-keys

Send keystrokes to a pane. Use \n for Enter.

mosaic pane send-keys --keys "ls\n"
mosaic pane send-keys --keys "npm test\n" --pane_id "A1B2C3D4-..."
ParamTypeRequiredDescription
--keysstringyesKeystrokes to send
--pane_idUUIDnoTarget pane (defaults to focused)

pane read-screen

Read the visible text content of a pane.

mosaic pane read-screen
mosaic pane read-screen --pane_id "A1B2C3D4-..."
ParamTypeRequiredDescription
--pane_idUUIDnoTarget pane (defaults to focused)

surface.*

Lower-level surface operations. Surfaces map to panels in Mosaic's domain model. No surface command steals focus.

surface list

List all surfaces across workspaces.

mosaic surface list
mosaic surface list --workspace_id "A1B2C3D4-..."

surface send_text

Send text to a surface.

mosaic surface send_text --text "hello" --surface_id "A1B2C3D4-..."

surface send_key

Send a named key to a surface (e.g., "enter", "tab", "escape").

mosaic surface send_key --key "enter" --surface_id "A1B2C3D4-..."

notification.*

Manage in-app notifications. No notification command steals focus.

notification create

Create a new notification.

mosaic notification create --title "Build done" --body "All tests passed"
mosaic notification create --title "Error" --workspace_id "A1B2C3D4-..."
ParamTypeRequiredDescription
--titlestringyesNotification title
--bodystringnoNotification body
--workspace_idUUIDnoScope to workspace
--pane_idUUIDnoScope to pane

notification list

List all notifications, optionally filtered by workspace.

mosaic notification list
mosaic notification list --workspace_id "A1B2C3D4-..."

notification clear

Clear a specific notification or all notifications.

mosaic notification clear --id "A1B2C3D4-..."  # clear one
mosaic notification clear                        # clear all
mosaic notification clear --workspace_id "..."   # clear all in workspace

notification subscribe

Subscribe to notification events via the EventBus.

mosaic notification subscribe

notification unsubscribe

Unsubscribe from notification events.

mosaic notification unsubscribe --token "..."

system.*

Read-only system information commands.

system ping

Health check. Returns pong: true.

mosaic system ping

system version

Returns the Mosaic version, name, and platform.

mosaic system version

system status

Returns status, version, and process ID.

mosaic system status

system capabilities

Lists all registered RPC methods and the app version.

mosaic system capabilities
mosaic system capabilities --json

ai.*

Interact with Mosaic AI, the natural language control plane.

ai message

Send a natural language message. Mosaic AI interprets the intent, maps to socket commands, safety-checks, and executes.

mosaic ai message --content "split pane and run npm test"
mosaic ai message --content "show me all workspaces"
mosaic ai message --content "what is running in pane 2?" --session_id "..."
ParamTypeRequiredDescription
--contentstringyesNatural language message
--session_idUUIDnoConversation session (defaults to active workspace)

ai status

Check AI engine status: provider, model, readiness, and processing state.

mosaic ai status

ai history

Retrieve conversation history for a session.

mosaic ai history
mosaic ai history --session_id "..." --limit 20
ParamTypeRequiredDescription
--session_idUUIDnoSession to query (defaults to active workspace)
--limitintnoMax turns to return (default 50, max 200)
--before_idstringnoCursor for pagination

browser.*

Control embedded browser panels. Limited to 7 basic navigation commands for v1.

browser navigate

mosaic browser navigate --url "https://example.com"

browser back / forward / reload

mosaic browser back
mosaic browser forward
mosaic browser reload

browser get_url / get_title

mosaic browser get_url
mosaic browser get_title

browser execute_js

mosaic browser execute_js --script "document.title"
ParamTypeRequiredDescription
--scriptstringyesJavaScript to evaluate
--surface_idUUIDnoTarget browser panel

Examples

Scripting a development workflow

# Create a workspace and split it
WS=$(mosaic workspace create --title "Dev" --json | python3 -c \
    "import sys,json; print(json.load(sys.stdin)['result']['id'])")

mosaic workspace select --id "$WS"
mosaic pane split --direction vertical

# Run server in left pane, tests in right pane
mosaic pane send-keys --keys "npm start\n"
# (focus switches to new pane after split)
mosaic pane send-keys --keys "npm test --watch\n"

Reading terminal output

# Get the text visible in the current pane
mosaic pane read-screen --json

# Read from a specific pane
mosaic pane read-screen --pane_id "A1B2C3D4-..."

Using AI for natural language control

# Split and run a command
mosaic ai message --content "split the pane horizontally and run pytest"

# Query workspace state
mosaic ai message --content "how many workspaces do I have?"

# Check what is running
mosaic ai message --content "what is the output in the right pane?"
