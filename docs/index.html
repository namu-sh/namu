Mosaic

Mosaic is a native macOS terminal multiplexer built on Ghostty with GPU-accelerated Metal rendering. It organizes work into workspaces with split panes, exposes full programmatic control via a JSON-RPC socket API, and provides Mosaic AI -- a built-in natural language control plane accessible through Telegram.

Features

GPU-Accelerated Terminal

Ghostty-powered Metal rendering with sub-5ms p99 typing latency. Full color support, Unicode, CJK characters, and image protocols.

Workspaces and Splits

Tabbed workspaces with arbitrary horizontal and vertical split nesting. Drag-and-drop reordering. Keyboard-driven navigation.

Shell Integration

Tracks working directory, git branch, listening ports, and command exit codes. OSC 133 semantic zones for prompt detection.

Session Persistence

Quit and relaunch restores all workspaces, split layouts, and scrollback content. Autosaves every 8 seconds.

Socket API

JSON-RPC 2.0 over Unix domain socket. Full programmatic control of workspaces, panes, surfaces, notifications, and AI.

CLI Tool

The mosaic command-line tool for scripting. Workspace management, pane operations, key injection, screen reading.

Mosaic AI

Natural language control plane. Send "split pane and run npm test" and it happens. Claude or OpenAI as swappable backends.

Command Safety

Three-tier safety classification. Destructive pattern detection. Rate limiting for external sources. Confirmation for dangerous commands.

Telegram Gateway

Receive build failure alerts on your phone. Reply with natural language commands. Pairing-token authentication with HMAC signing.
