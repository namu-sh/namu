Namu

Namu is a native macOS terminal multiplexer built on Ghostty. It combines split-pane terminal workflows, browser automation, a JSON-RPC socket API, a scriptable CLI, and a built-in natural-language control plane called NamuAI.

The current checkout ships local AI control, outbound alert routing, and an authenticated TCP relay for remote forwarding. Older references to a standalone gateway are historical and are not part of the active build target.

Features

Ghostty Terminal

GPU-accelerated rendering, low-latency input, Unicode support, copy mode, image transfer, and shell integration.

Workspaces and Splits

Tabbed workspaces with arbitrary horizontal and vertical pane trees powered by Bonsplit.

JSON-RPC Socket API

Full programmatic control over workspaces, panes, surfaces, notifications, browser panels, and AI.

CLI and Hooks

The namu CLI supports scripting, tmux compatibility, Claude teams, and Codex hooks.

NamuAI

A built-in NL control plane that interprets intent, maps it to structured commands, safety-checks them, and executes them.

Embedded Browser

Browser panels expose navigation, DOM querying, JavaScript evaluation, screenshots, and higher-level automation commands.

Alert Routing

Rule-based alert detection with outbound Slack, Telegram, Discord, and generic webhook delivery.

Remote Relay

An authenticated TCP relay forwards JSON-RPC requests without exposing the local Unix socket directly.

Native macOS UI

SwiftUI + AppKit views for sidebar, command palette, workspace rendering, AI chat, notifications, and updates.
